# 📈 AI Stock Market Dashboard

An AI-powered Stock Market Analytics Dashboard built using Python, Streamlit, Yahoo Finance, and Machine Learning.

## 🚀 Features

* Real-time stock market data
* Candlestick charts
* Moving Averages (MA50 & MA200)
* RSI Indicator
* MACD Indicator
* Bollinger Bands
* Portfolio Tracker
* Profit/Loss Calculator
* Risk Analysis
* News Sentiment Analysis
* AI Stock Score
* AI Recommendation Engine
* AI Price Prediction
* CSV Report Download

## 🛠️ Technologies Used

* Python
* Streamlit
* Yahoo Finance (yfinance)
* Pandas
* NumPy
* Plotly
* TA-Lib / ta
* TextBlob
* Feedparser
* TensorFlow / Scikit-Learn

## ⚙️ Installation

```bash
git clone <repository-url>
cd AI-Stock-Market-Dashboard

pip install -r requirements.txt

streamlit run app.py
```
